[SerializeField]란?

  • UnityEngine class에서 제공하는 attribute
    . attribute란?
    … metadata 정보를 assembly에 추가할 수 있도록 해주는 C# object
  • unity는 기본적으로 public field는 serialize 하나,
    private field도 serialize 하고 싶다면, [SerializeField] attribute를 추가해야 함.
    https://docs.unity3d.com/ScriptReference/SerializeField.html
  • 보통 unity내 inspector 창에서 script의 private 값을 변경하고자 할 때 많이 사용함.
    https://forum.unity.com/threads/when-to-use-serializefield-and-why.184687/
