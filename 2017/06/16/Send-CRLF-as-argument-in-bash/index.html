bash에서 CRLF 값을 argument로 넘겨주기

shellcode를 argument로 넘길 때,
shellcode 내에 0xa(LF), 0xd(CR) 값이 포함되어 있다면,
bash는 이를 command line 마지막 부분으로 해석하여,
그 바로 전까지만 argument로 넘기고,
그 뒤 shellcode는 argument로 넘어가지 않습니다.

이와 같은 동작에 제한을 받지 않고,
CRLF 값도 argument로 넘기기 위해
bash에게 특수문자를 문자 그대로 받아들이라고 일러주는 방법입니다.
