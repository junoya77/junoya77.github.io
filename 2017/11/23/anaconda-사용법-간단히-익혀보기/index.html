anaconda 사용법 간단히 익혀보기

WSL(Windows Subsystem Linux) + Windows 10 환경에서 anaconda를 설치해 봅니다.

먼저 설치 프로그램을 내려받습니다.

설치 프로그램을 실행합니다.

anaconda 사용법을 간단히 익혀봅니다.

  • 참고 : https://conda.io/docs/user-guide/getting-started.html

anaconda 명령어 PATH 적용을 위해,
bash shell을 exit 한 후, bash shell을 다시 시작합니다.

설치된 anaconda 버전을 확인해 봅니다.

ananconda를 최신 버전으로 업데이트 해 봅니다.

“snowflakes”라는 이름의 가상환경을 만들어 봅니다.

이때 “snowflakes” 환경을 만들면서 biopython package 도 그 가상환경에 함께 설치합니다.
biopython은 bioinformatics 분야에서 활용되는 python package라고 합니다.

  • 참고 : http://biopython.org/

“snowflakes” 가상환경이 잘 생성되었는지 확인해 봅니다.

“snowflakes” 가상환경으로 진입합니다.

“snowflakes” 가상환경을 빠져나갑니다.

“snowflakes” 가상환경을 삭제합니다.

“snowflakes” 가상환경이 삭제되었는지 확인해 봅니다.

anaconda에 기본적으로 설치된 package 목록을 확인해 봅니다.

package 설치 실습을 위해 “test”라는 이름의 가상환경을 생성합니다.

“test” 가상환경이 생성되었는지 확인해 봅니다.

“test” 가상환경으로 진입합니다.

“test” 가상환경에 설치된 package 목록을 확인해 봅니다.
설치된 package가 하나도 없다는 걸 확인할 수 있습니다.

“test” 가상환경을 빠져나갑니다.

“test” 가상환경에 beautifulsoup4 package를 설치합니다.

package가 설치되었는지 확인하기 위해 “test” 가상환경으로 진입합니다.

“test” 가상환경에서 package 설치 목록을 조회해 봅니다.
beautifulsoup4 package 뿐만 아니라,
dependency package들도 함께 설치되었음을 확인할 수 있습니다.
